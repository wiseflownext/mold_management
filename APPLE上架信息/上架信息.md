## 三、App Store 描述

### 宣传文本（170字以内，可随时修改，无需审核）

```
企业级模具管理解决方案，实时追踪模具状态、使用次数与保养周期，智能提醒维保计划，让模具管理更高效。
```

### 描述（4000字以内）

```
模具管理APP是一款专为制造业企业设计的模具全生命周期管理工具，帮助企业高效管理模具资产。

【核心功能】
• 模具台账：记录模具基本信息、设计寿命、模腔数、关联产品等，支持按车间、状态筛选
• 使用记录：每次使用自动累计产出数量，实时追踪模具使用寿命消耗
• 维保管理：记录每次维修保养详情，支持拍照上传维保现场照片
• 智能提醒：按次数保养、定期保养双维度提醒，到期自动通知相关人员
• 数据看板：首页展示在用/维修/停用/报废模具数量，今日使用概览一目了然
• 数据报表：支持按车间、时间维度导出使用和维保报表
• 多角色权限：管理员与普通员工差异化功能，数据安全有保障

【适用场景】
注塑、冲压、压铸等制造业企业的模具资产管理

【产品优势】
• 操作简单，一线员工快速上手
• 数据实时同步，多人协作无障碍
• 维保到期智能提醒，避免漏保
• 完整操作日志，管理可追溯
```
