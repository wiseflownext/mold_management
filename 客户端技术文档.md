## 一、技术选型

| 项目 | 选型 | 说明 |
|------|------|------|
| 框架 | Flutter 3.x | Dart 语言，跨平台一套代码 |

| 图片缓存 | cached_network_image | OSS 图片加载 |

| 本地构建 | Flutter CLI | flutter build apk / ipa |
| 云构建 | GitHub Actions + Codemagic | CI/CD 自动化 |

| 下拉刷新 | pull_to_refresh_flutter3 | 列表刷新/加载更多 |

## 三、API 对接

### 3.1 基础地址

```
生产环境: http://121.40.172.33:8088/api
```

### 3.2 统一响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 3.3 认证流程

```
1. POST /auth/login  → 获取 accessToken + refreshToken
2. 请求头附加: Authorization: Bearer {accessToken}
3. 401 时自动用 refreshToken 刷新
4. refreshToken 也过期 → 跳转登录页
```

## 六、核心模块设计

### 6.1 JWT 认证拦截器

```
Dio 请求拦截器:
  → 自动附加 Authorization: Bearer {accessToken}

Dio 错误拦截器:
  → 401 → 用 refreshToken 调 /auth/refresh
    → 成功 → 替换 token → 重试原请求
    → 失败 → 清除登录态 → 跳转 /login
```

### 6.2 分页加载

```
通用分页 Provider:
  - page: 当前页码
  - hasMore: 是否有更多
  - items: 已加载数据
  - loadMore(): 加载下一页
  - refresh(): 重置并重新加载
```

### 6.3 图片上传流程

```
拍照/相册 → image_picker
  → 压缩 → flutter_image_compress (≤1MB)
  → multipart/form-data → POST /api/upload/image
  → 返回 OSS URL
```

### 6.4 CSV 导出

```
GET /reports/usage → Response bytes (CSV)
  → path_provider 获取临时目录
  → 写入文件
  → share_plus 分享/保存
```
