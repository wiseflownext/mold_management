# 模具管理 APP 产品功能文档

## 一、产品概述

| 项目 | 说明 |
|------|------|
| 产品名称 | 模具管理 |
| 行业 | 注塑制造业 |
| 定位 | 工厂模具全生命周期管理平台 |
| 目标平台 | iOS / Android（后续双端发布） |
| 部署方式 | 云端混合部署，暂不支持离线 |
| UI语言 | 中文 |
| 设计尺寸 | iPhone 15 Pro（393×852） |

### 设计规范

| 项目 | 值 |
|------|-----|
| 主色 | #1A73E8（蓝） |
| 强调色 | #FF6D00（橙，用于告警/提醒） |

## 三、数据模型

### 3.1 模具（Mold）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| moldNumber | string | 模具编号，如 `5705-JL153-1` |
| type | enum | 模具类型：`模压模具` / `口型模具` / `接角模具` |
| workshop | string | 所属车间（可选） |
| firstUseDate | string | 首次使用日期 |
| designLife | number | 设计寿命（次） |
| maintenanceCycle | number | 保养周期（次） |
| usageCount | number | 累计使用次数 |
| status | enum | 状态：`在用` / `维修中` / `停用` / `报废` |
| products | Product[] | 关联产品列表 |
| usageRecords | UsageRecord[] | 使用记录 |
| maintenanceRecords | MaintenanceRecord[] | 维保记录 |
| certificationReport | string | 鉴定报告文件（用于延寿） |

### 3.2 关联产品（Product）

| 字段 | 类型 | 说明 |
|------|------|------|
| customer | string | 客户名称 |
| model | string | 车型 |
| name | string | 产品名称 |
| partNumber | string | 零件号 |

### 3.3 使用记录（UsageRecord）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| date | string | 日期 |
| product | string | 关联产品名称 |
| quantity | number | 生产数量（次） |
| operator | string | 操作人 |
| shift | enum | 班次：`早班` / `中班` / `晚班` |
| note | string | 备注（选填） |

### 3.4 维保记录（MaintenanceRecord）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| date | string | 日期 |
| type | enum | 维保类型：`保养` / `维修` |
| content | string | 维保内容描述 |
| operator | string | 操作人 |

### 3.5 员工账号（Employee）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| username | string | 登录账号 |
| name | string | 姓名 |
| password | string | 密码（加密存储） |
| role | enum | 角色：`admin` / `operator` |
| workshop | string | 所属车间（可选） |
| createdAt | string | 创建时间 |
| lastLogin | string | 最后登录时间 |

### 3.6 通知（Notification）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| type | enum | 类型：`maintenance_overdue` / `maintenance_soon` / `status_change` / `usage_record` |

| message | string | 通知内容 |
| time | string | 时间 |
| isRead | boolean | 是否已读 |

### 3.7 维保提醒配置（ReminderSetting）

| 字段 | 类型 | 说明 |
|------|------|------|
| moldType | enum | 模具类型 |
| enabled | boolean | 是否启用提醒 |
| remainingThreshold | number | 剩余次数阈值 |
| warningPercent | number | 预警百分比（如80%） |
| overduePercent | number | 逾期百分比（如100%） |

## 四、功能模块详细说明

### 4.1 登录页

**路由：** `/login`

**功能描述：**
- 手机号 + 密码登录，无注册功能
- 支持密码显示/隐藏切换
- 提供快速登录入口（演示用，正式版可移除）
- 登录成功后跳转首页，登录状态持久化（localStorage）

**表单字段：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| 手机号 | tel | ✓ | 手机号输入 |
| 密码 | password | ✓ | 支持 Enter 键提交 |

**交互：**
- 登录成功 → 跳转 `/home`
- 登录失败 → Toast 提示"手机号或密码错误"
- 空字段提交 → Toast 提示"请填写手机号和密码"

---

### 4.2 首页

**路由：** `/home`  
**Tab位置：** 底部导航第1个

**功能描述：**

#### 4.2.1 顶部 Header
- 显示当前用户姓名、角色标签（如"管理员"）
- 显示当前日期（X月X日 周X）
- 右侧通知铃铛按钮，有未读时显示橙色圆点

#### 4.2.2 快捷操作
- 两个大按钮并排：
  - **新增使用记录**（蓝色）→ 跳转 `/add-usage`
  - **新增维保记录**（橙色）→ 跳转 `/add-maintenance`

#### 4.2.3 模具状态统计
- 2×2 网格展示四种状态的模具数量：
  - 在用（绿）、维修中（橙）、停用（灰）、报废（红）

#### 4.2.4 维保提醒
- 展示待保养的模具列表（最多显示3条）
- 每条显示：模具编号、产品名称、距下次保养剩余次数
- 逾期项以红色高亮，显示"已逾期 X 次"
- 每条附带迷你进度条
- 点击单条 → 跳转该模具详情
- "查看全部" → 跳转 `/reminders`

#### 4.2.5 今日生产概览
- 三个指标：使用记录数（条）、生产总量（次）、活跃模具数（副）

#### 4.2.6 消息通知面板
- 点击铃铛 → 底部弹出通知面板
- 通知类型：维保逾期提醒、保养提醒、状态变更、新增使用记录
- 每条通知显示：图标、标题、内容、时间、未读标记
- 支持"全部已读"
- 点击通知 → 关闭面板并跳转关联模具详情

---

### 4.3 模具列表

**路由：** `/molds`  
**Tab位置：** 底部导航第2个

**功能描述：**

#### 4.3.1 搜索
- 顶部搜索栏，支持按模具编号、产品名称、客户名称模糊搜索

#### 4.3.2 分类筛选
- Segment Tab 切换：全部 / 模压模具 / 口型模具 / 接角模具
- 筛选条件：
  - 状态下拉：全部 / 在用 / 维修中 / 停用 / 报废
  - 客户下拉：全部 / 各客户名称
- 显示筛选结果计数（如"共 8 副"）

#### 4.3.3 模具卡片列表
- 每张卡片显示：
  - 左侧：模具类型图标（口型/模压/接角，不同颜色）
  - 中间：模具编号（加粗）、产品名称、客户名称
  - 右侧：状态标签（彩色药丸）、已使用次数、箭头
- 点击卡片 → 跳转 `/molds/:id`

#### 4.3.4 新增模具按钮（仅管理员可见）
- Header 右侧"新增模具"按钮 → 跳转 `/add-mold`

---

### 4.4 模具详情

**路由：** `/molds/:id`

**功能描述：**

#### 4.4.1 基本信息卡
- 模具编号（大字）、状态标签、所属车间
- 信息网格（2列）：
  - 类型、车间、首次使用日期、保养周期、已使用次数、设计寿命
- 设计寿命被修改后显示"已更新"标签

#### 4.4.2 寿命消耗进度条
- 百分比显示寿命消耗（如 24.7%）
- 颜色变化：≤60% 蓝色、60%-80% 黄色、>80% 红色
- 底部显示：0 / 距下次保养还剩 X 次 / 设计寿命

#### 4.4.3 关联产品
- 展示该模具关联的所有产品
- 每个产品显示：客户、车型、产品名称、零件号

#### 4.4.4 使用记录 / 维保记录 Tab
- 两个 Tab 切换
- **使用记录**：时间线样式，每条显示日期、班次标签、产品名、产量、操作人
- **维保记录**：时间线样式，每条显示日期、维保类型标签（保养/维修）、内容、操作人
- 底部"新增维保记录"快捷入口

#### 4.4.5 管理员操作（仅管理员可见）
- **鉴定**按钮 → 打开鉴定报告上传面板
- **变更**按钮 → 打开状态变更底部弹窗
- **删除**按钮 → 打开删除确认弹窗

#### 4.4.6 鉴定报告上传（管理员）
- 底部弹窗形式
- 上传鉴定报告图片（支持 JPG/PNG/HEIC）
- 上传后可填写新的设计寿命
- 显示当前设计寿命和变更差值预览
- 操作提示：修改需上传凭证，系统留存记录
- 管理员直接确认生效，无需审批

#### 4.4.7 状态变更（管理员）
- 底部弹窗显示当前状态
- 可选择切换到其他状态（排除当前状态）
- 点击即变更，Toast 提示成功

#### 4.4.8 删除模具（管理员）
- 二次确认弹窗，提示数据不可恢复
- 确认后删除并跳转回模具列表

### 4.5 新增使用记录

**路由：** `/add-usage`  
**Tab位置：** 底部导航第3个（直接进入该页面）

**功能描述：**

**表单字段：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| 模具编号 | 搜索选择器 | ✓ | 支持输入搜索，只显示"在用"状态的模具 |
| 关联产品 | 下拉选择 | ✓ | 选模具后自动加载该模具的产品列表 |
| 生产数量 | 数字输入 + 步进器(±10) | ✓ | 单位：次 |
| 班次 | 单选按钮组 | ✓ | 早班 / 中班 / 晚班 |
| 日期 | 日期选择器 | ✗ | 默认当天 |
| 操作人 | 自动填充 | — | 当前登录用户，不可编辑 |
| 备注 | 多行文本 | ✗ | 选填 |

**交互：**
- 选择模具 → 自动加载关联产品并默认选中第一个
- 提交校验：模具编号、产品、数量>0 必填
- 提交成功 → 显示成功动画（绿色勾）→ 1.5秒后自动返回上一页
- 提交中 → 按钮显示"提交中..."并禁用

### 4.6 新增维保记录

**功能描述：**

**表单字段：**

### 4.10 维保提醒列表

**路由：** `/reminders`

**功能描述：**

#### 4.10.1 统计概览
- 顶部橙色 Header，显示逾期/紧急/预警/正常四个计数

#### 4.10.2 筛选 Tab
- 全部 / 逾期 / 即将到期 / 正常

#### 4.10.3 紧急度分级

| 等级 | 条件 | 颜色 |
|------|------|------|
| 逾期(overdue) | 剩余次数 < 0 | 红色 |
| 紧急(critical) | 剩余次数 ≤ 200 | 橙色 |
| 预警(warning) | 剩余次数 ≤ 500 | 黄色 |
| 正常(normal) | 剩余次数 > 500 | 绿色 |

#### 4.10.4 提醒卡片
- 每张卡片显示：
  - 紧急度标签 + 上次保养日期
  - 模具编号、模具类型标签、产品名称
  - 距下次保养剩余/逾期次数
  - 保养周期进度条
  - 车间、累计使用次数
  - 管理员可见"安排维保"按钮 → 跳转新增维保（带入模具编号）
- 点击卡片 → 跳转模具详情

### 4.11 维保提醒设置（仅管理员）

**路由：** `/reminder-settings`

**功能描述：**

#### 4.11.1 模具类型专属配置
- 为每种模具类型（模压/口型/接角）独立配置：

| 配置项 | 类型 | 说明 |
|--------|------|------|
| 启用开关 | toggle | 是否启用该类型的提醒 |
| 剩余次数阈值 | 数字输入 | 低于此值时触发提醒（10~10000） |
| 预警百分比 | 滑块 | 使用率达到此值时预警（50%~95%） |
| 逾期百分比 | 滑块 | 超过此值时逾期告警（95%~120%） |

- 每种类型下方显示触发规则总结

#### 4.11.2 通知方式
- App 内推送通知（首页铃铛图标）开关
- 每日汇总推送（每天 08:00 发送）开关

### 4.13 车间管理（仅管理员）

**路由：** `/workshop-management`

**功能描述：**
- 预设车间：挤出车间、后续车间（标记为"默认"，不可删除，可编辑名称）
- 支持新增车间：点击"添加新车间" → 弹窗输入名称
- 支持编辑车间名称（行内编辑，Enter保存，Escape取消）
- 支持删除非默认车间
- 车间名称不可重复

### 4.14 数据报表导出（仅管理员）

**路由：** `/data-report`

**功能描述：**

#### 报表类型

| 报表 | 说明 | 包含字段 |
|------|------|---------|
| 使用记录报表 | 导出模具使用记录 | 模具编号、产品名称、客户、车间、日期、数量、单位、操作员、班次 |
| 维保记录报表 | 导出维修保养记录 | 模具编号、模具类型、车间、日期、类型、内容、操作员 |

- 导出前显示数据预览（条数/包含字段）

### 4.15 修改密码

**路由：** `/change-password`

**功能描述：**

**表单字段：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| 原密码 | 密码输入 | ✓ | 至少6位 |
| 新密码 | 密码输入 | ✓ | 至少8位 |
| 确认新密码 | 密码输入 | ✓ | 需与新密码一致 |

- 所有密码字段支持显示/隐藏切换
- 新密码强度检测（弱/中/强/很强），带可视化强度条
- 密码要求实时校验：至少8字符、包含数字、包含字母、与原密码不同
- 两次密码不一致时红色提示
- 修改成功 → 成功动画 → 自动返回
